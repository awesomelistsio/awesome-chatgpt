> A curated list of tools, extensions, prompts, projects, and resources for ChatGPT and the OpenAI ecosystem.

ChatGPT is an AI chatbot developed by OpenAI, based on large language models like GPT-3.5 and GPT-4. This list includes everything from productivity tools to creative applications, helping users get more out of ChatGPT.

## Prompt Collections

- **[Awesome ChatGPT Prompts](https://github.com/f/awesome-chatgpt-prompts)** – Curated list of high-quality, ready-to-use ChatGPT prompts.
- **[FlowGPT](https://flowgpt.com/)** – Discover and share top ChatGPT prompts from the community.
- **[PromptBase](https://promptbase.com/)** – Marketplace for buying and selling effective prompts.
- **[PromptHero](https://prompthero.com/)** – Prompts for ChatGPT, Midjourney, and more.

## Browser Extensions

- **[ChatGPT for Google](https://chrome.google.com/webstore/detail/chatgpt-for-google/)** – Show ChatGPT responses alongside search engine results.
- **[WebChatGPT](https://github.com/qunash/webchatgpt)** – Adds web search results to ChatGPT prompts.
- **[Merlin](https://merlin.foyer.work/)** – ChatGPT assistant in your browser and productivity tools.
- **[ChatGPT File Uploader](https://github.com/wong2/chat-gpt-google-extension)** – Upload files directly to ChatGPT sessions.

## APIs & Wrappers

- **[OpenAI API](https://platform.openai.com/docs/api-reference/introduction)** – Official API for GPT models and ChatGPT functions.
- **[ChatGPT API Wrapper (Unofficial)](https://github.com/transitive-bullshit/chatgpt-api)** – Reverse-engineered Node.js wrapper for ChatGPT web interface.
- **[LangChain](https://www.langchain.com/)** – Framework to build apps with ChatGPT and other LLMs.
- **[LlamaIndex](https://www.llamaindex.ai/)** – Framework for building context-aware LLM applications.

## Developer Tools

- **[PromptLayer](https://www.promptlayer.com/)** – Track, manage, and version LLM prompts.
- **[Promptfoo](https://promptfoo.dev/)** – Evaluate and compare prompt performance programmatically.
- **[Auto-GPT](https://github.com/Torantulino/Auto-GPT)** – Autonomous GPT-4 agent that chains thoughts using prompts.
- **[AgentGPT](https://agentgpt.reworkd.ai/)** – Deploy autonomous AI agents directly in the browser.

## ChatGPT Apps & Clients

- **[Raycast ChatGPT Extension](https://www.raycast.com/openai/chatgpt)** – ChatGPT integration in Raycast launcher.
- **[ChatGPT Desktop App (Unofficial)](https://github.com/lencx/ChatGPT)** – Cross-platform desktop app for ChatGPT.
- **[ChatGPT Writer](https://chatgptwriter.ai/)** – Compose emails and messages using ChatGPT from your browser.

## ChatGPT Use Cases

- **[AI Dungeon](https://play.aidungeon.io/)** – AI-powered interactive storytelling game.
- **[Elicit](https://elicit.org/)** – Research assistant that uses LLMs to automate scientific literature reviews.
- **[YouTube Summary with ChatGPT](https://glasp.co/youtube-summary)** – Summarize YouTube videos using ChatGPT.
- **[ChatGPT for Slides](https://www.tome.app/)** – Generate slide decks and visual storytelling with AI assistance.
